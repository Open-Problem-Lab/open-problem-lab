# Outputs

## Accepted Outputs

No accepted outputs exist yet.

This is deliberate. The pack is scoped, not solved.

## Candidate Output Types

- Evidence and data source inventory with access status.
- Breeding-habitat suitability baseline maps.
- Outbreak signal correlation analysis with lead-time distributions.
- Swarm trajectory hindcast validation.
- Failure-mode review identifying forecast misuse risks.
- Field-reality review with control-operation context.

## Publication Rule

Published outputs must say what they are not. A locust outbreak risk map is not a control operation order, pesticide recommendation, or crop-loss forecast.
