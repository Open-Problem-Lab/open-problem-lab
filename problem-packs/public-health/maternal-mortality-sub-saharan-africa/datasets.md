# Dataset Inventory

| Source                     | Grain    | Status           | Use                                         |
| -------------------------- | -------- | ---------------- | ------------------------------------------- |
| DHS delivery data          | Cluster  | Usable           | Facility delivery rates, place of delivery  |
| MAP travel-time surfaces   | 1km      | Usable           | Travel time to nearest health facility      |
| Health facility registries | Facility | Variable         | Facility location and EmOC signal functions |
| SPA facility surveys       | Facility | Usable, periodic | Facility readiness assessment               |
| WorldPop population        | 100m     | Usable           | Population within access catchments         |

## Required Properties

Facility location, EmOC signal function data, travel-time methodology, license.

## Rejection Rule

Rejected if facility data currency or methodology unverifiable.
