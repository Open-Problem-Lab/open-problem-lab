# Dataset Inventory

| Source                   | Grain    | Status                 | Use                            |
| ------------------------ | -------- | ---------------------- | ------------------------------ |
| CHIRPS rainfall          | 0.05 deg | Usable                 | Rainfall deficit detection     |
| MODIS NDVI               | 250m-1km | Usable                 | Vegetation stress              |

## Required Properties

Date range, geographic grain, temporal resolution, classification methodology, license.

## Rejection Rule

Rejected if methodology or grain unverifiable.
