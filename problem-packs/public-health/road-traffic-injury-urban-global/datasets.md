# Dataset Inventory

| Source                     | Grain           | Status           | Use                            |
| -------------------------- | --------------- | ---------------- | ------------------------------ |
| Police crash reports       | Point, variable | Usable with gaps | Crash location and severity    |
| Hospital trauma registries | Facility        | Variable         | Injury outcome data            |
| OpenStreetMap roads        | Building/block  | Usable           | Road network and crash factors |
| Health facility locations  | Facility        | Variable         | Trauma care locations          |
| WorldPop population        | 100m            | Usable           | Population at risk             |

## Required Properties

Crash location, severity classification, trauma facility capability, travel-time methodology, license.

## Rejection Rule

Rejected if crash-location methodology or trauma-capability data unverifiable.
