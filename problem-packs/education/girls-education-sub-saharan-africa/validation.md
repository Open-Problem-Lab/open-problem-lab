# Validation

| Layer           | Gate                    |
| --------------- | ----------------------- |
| Structure       | `pnpm validate`         |
| Evidence        | Dated records           |
| Reproducibility | Rerunnable              |
| Review          | Approvers               |
| Replication     | Quantitative replicated |

## Baseline Requirements

Enrollment gaps must document: survey source and year, sample design, gender disaggregation, confidence intervals.

## Model Requirements

No barrier classification validated without: cited evidence for each barrier, evidence strength rating, replication.
