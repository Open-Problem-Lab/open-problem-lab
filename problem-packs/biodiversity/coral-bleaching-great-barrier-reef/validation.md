# Validation

| Layer           | Gate                           |
| --------------- | ------------------------------ |
| Structure       | `pnpm validate`                |
| Evidence        | Dated records                  |
| Reproducibility | Rerunnable steps               |
| Review          | Required approvers             |
| Replication     | Quantitative claims replicated |

## Baseline Requirements

Bleaching validation must document: DHW threshold source, in-water survey methodology, comparison statistics, spatial coverage gaps.

## Model Requirements

No severity map validated without: satellite-survey comparison, methodology documentation, coverage gap identification, replication.
