# Dataset Inventory

| Source                  | Grain    | Status             | Use                           |
| ----------------------- | -------- | ------------------ | ----------------------------- |
| FAOSTAT crop statistics | Country  | Usable for framing | National yield trends         |
| Global Yield Gap Atlas  | ~10km    | Usable             | Water-limited yield potential |
| Sentinel-2 NDVI         | 10m      | Usable             | Satellite yield proxy         |
| AfSIS soil data         | ~250m    | Usable             | Soil constraints              |
| CHIRPS rainfall         | 0.05 deg | Usable             | Rainfall and drought context  |

## Required Properties

Crop type, yield metric, estimation methodology, soil data source, license.

## Rejection Rule

Rejected if yield estimation methodology unverifiable.
