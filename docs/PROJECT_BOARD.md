# Project Board

Use one GitHub Project board for v0.

## Columns

```text
Inbox
Needs Triage
Scoped
Agent Working
Needs Review
Needs Replication
Accepted
Published
Field-Tested
```

## Required Fields

- Problem ID
- Type
- Domain
- Region
- Risk
- Reviewer Needed
- Done Condition
- Linked Pull Request

## Triage Rule

If an issue lacks a done condition, it goes back to Discussion or is closed as unscoped.
