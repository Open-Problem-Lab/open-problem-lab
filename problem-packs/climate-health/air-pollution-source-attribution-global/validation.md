# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national source-attribution analyses must document:

- Apportionment method used (chemical tracer, PMF, CTM) with methodology-specific limitations and inter-method variance documentation.
- Ground-monitor density and population-representativeness assessment for the analyzed city.
- Satellite PM2.5 validation status and uncertainty bounds (20-50 percent for LMIC cities).
- Seasonal variation in source contributions — document whether the analysis covers representative seasons.
- Distinction between published apportionment data from the analyzed city versus borrowed attribution from a different city context.

## Source-Attribution Requirements

No source-attribution claim may be called actionable until it includes:

- Apportionment-methodology documentation with explicit statement of how results would differ under alternative methods.
- Monitor-representativeness assessment — document whether the limited number of monitoring sites represents city-wide source contributions.
- Comparison with at least one published source-apportionment study for a similar LMIC megacity context, with explicit transferability assessment.
- Sensitivity analysis showing how attribution percentages change under alternative apportionment methods or monitor subsets.
- LMIC megacity evidence-gap documentation stating what monitoring and apportionment infrastructure would be needed for definitive attribution.

## Sector Health-Impact Requirements

No source-specific health-impact claim may be merged without:

- Concentration-response function transferability documentation — state whether the function is from the analyzed LMIC context or transferred from high-income studies.
- Uncertainty propagation from source-attribution estimates through to health-impact estimates.
- Documentation that source-attribution percentages are methodology-dependent and may change under alternative methods.
- Non-use case documentation (not for source-specific industrial closure without local apportionment evidence, not for borrowed-context policy without transferability validation).
