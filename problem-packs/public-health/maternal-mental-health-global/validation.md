# Validation

| Layer           | Gate                    |
| --------------- | ----------------------- |
| Structure       | `pnpm validate`         |
| Evidence        | Dated records           |
| Reproducibility | Rerunnable              |
| Review          | Approvers               |
| Replication     | Quantitative replicated |

## Baseline Requirements

Screening data must document: instrument, validation status, population, consent procedures.

## Model Requirements

No detection gap validated without: screening-instrument documentation, consent safeguards, replication.
