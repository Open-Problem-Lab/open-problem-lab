# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national adoption-gap analyses must document:

- Seed-system data source (TASAI, national survey, CGIAR report) with date, crop-country coverage, and adoption-measurement methodology.
- Climate-resilient variety availability from CGIAR or national variety-release catalogs.
- On-farm yield-gap documentation — explicitly state the difference between on-station and on-farm performance for each crop-country combination.
- Farmer-preference documentation — identify which non-resistance traits (taste, storage, marketability) affect adoption decisions.
- Seed-system barrier classification methodology and tractability assessment.

## Adoption-Gap Requirements

No adoption-gap claim may be called actionable until it includes:

- Adoption-rate source and measurement methodology for the specific crop-country combination.
- Comparison with at least one published adoption study for a similar seed-system context.
- On-farm yield-gap documentation distinguishing on-station trial performance from farmer-field outcomes.
- Seed-system barrier classification identifying which bottlenecks (multiplication, distribution, awareness, affordability, trust) are most tractable.
- Sensitivity analysis showing how adoption-gap estimates change under alternative data sources or measurement methodologies.

## Seed-System Investment Requirements

No seed-system investment recommendation based on adoption-gap analysis may be merged without:

- Barrier-tractability assessment with farmer-preference context.
- Economic analysis of seed-subsidy crowding-out effects on private-sector distribution.
- Documentation that adoption does not guarantee yield advantage without complementary inputs and extension.
- Non-use case documentation (not for blanket seed-subsidy justification, not for variety-recommendation lists without local validation).
