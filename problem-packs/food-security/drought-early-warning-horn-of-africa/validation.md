# Validation

| Layer           | Gate                           |
| --------------- | ------------------------------ |
| Structure       | `pnpm validate`                |
| Evidence        | Dated records                  |
| Reproducibility | Rerunnable steps               |
| Review          | Required approvers             |
| Replication     | Quantitative claims replicated |

## Baseline Requirements

Signal validation must document: data sources, lead-time methodology, outcome classification, comparison statistics.

## Model Requirements

No food-insecurity signal validated without: lead-time distribution, false-alarm profiling, cost-of-error context, replication.
