# Playbooks

## Analyst Playbook

1. Start with `problem.json`, `evidence.json`, and `tasks.json`.
2. Select one scoped task.
3. Add evidence before adding claims.
4. For lab-field gap analysis: always document measurement methodology — laboratory tests use controlled conditions while field tests capture real-world degradation from fuel moisture, cooking practices, and stove maintenance.
5. For compliance assessment: always document verification-system coverage — fewer than 5 percent of deployed stoves have independent field verification in most countries.
6. For health-impact claims: always document whether performance data comes from laboratory or field measurements — laboratory data alone does not support field health-impact claims.
7. Mark every operational inference as reviewed, replicated, or blocked.

## Reviewer Playbook

Ask five questions:

1. What claim is being made about stove performance or standards compliance?
2. What source proves it — and is it laboratory test data, field-performance measurement, or compliance certification?
3. What method distinguishes laboratory from field performance, and what is the verification-coverage rate?
4. What would make it false — particularly if the 30-70 percent lab-field degradation applies, if compliance is self-certified without independent verification, or if health-impact claims are based on lab data alone?
5. Who could misuse it — particularly to claim health impact from laboratory performance data, to present compliance certification as field-verified performance, or to fund stove programs without field-verification requirements?

## Field-Reality Playbook

Do not ask whether a laboratory performance tier is impressive. Ask whether it would change a named decision for a named user at a useful time, without creating worse incentives. For clean cooking in LMICs, ask specifically:

- Would a Kenya Clean Cooking Association program director use this to set stove procurement standards, or would lab-performance data be treated as field-performance guarantee?
- Would a health ministry use this to justify clean-cooking investment based on emission-reduction claims, or would the 30-70 percent lab-field degradation make the actual health impact uncertain?
- Would a stove manufacturer use this for marketing claims, or would compliance certification without independent field verification create false consumer confidence?

The answer must include explicit non-use cases before any field-facing output is published.
