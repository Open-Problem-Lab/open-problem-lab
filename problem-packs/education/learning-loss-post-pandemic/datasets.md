# Dataset Inventory

## Candidate Sources

| Source                                   | Grain                           | Current status           | Use                                           |
| ---------------------------------------- | ------------------------------- | ------------------------ | --------------------------------------------- |
| World Bank Learning Poverty database     | Country                         | Usable for framing       | Aggregate learning poverty estimates          |
| UNESCO school closure tracker            | Country, weekly                 | Usable                   | Closure duration as primary exposure variable |
| PISA and TIMSS international assessments | Student/school, 3-4 year cycles | Usable, sparse in LICs   | Cross-country comparable learning outcomes    |
| Early Grade Reading Assessment data      | Student, irregular              | Restricted or fragmented | Country-specific learning measurement         |
| World Inequality Database on Education   | Country, annual                 | Usable                   | Completion and out-of-school rates            |

## Required Dataset Properties

- Assessment type, instrument, and language.
- Sample size and representativeness.
- Date of assessment administration.
- Learning domain and metric.
- Comparability notes for multi-year analysis.

## Rejection Rule

A dataset is rejected for canonical modeling if assessment type, date, sample, or comparability cannot be verified. Rejected datasets may still be listed as context.
