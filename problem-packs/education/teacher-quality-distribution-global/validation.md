# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national teacher-distribution analyses must document:

- EMIS data source with data-quality assessment — update frequency, completeness, and public accessibility.
- Teacher-qualification definition used — explicitly document what counts as 'trained' in each country and how definitions differ.
- SDI survey availability — document whether direct teacher-assessment data exists for the analyzed country.
- Cross-country comparability assessment — document harmonization methodology if comparing across countries.
- Rural-urban comparison methodology — document how districts are classified and what confound controls are applied.

## Deployment-Equity Requirements

No deployment-equity claim may be called actionable until it includes:

- Qualification-proxy limitation documentation — explicitly state that teacher qualification is a weak proxy for teaching effectiveness.
- Learning-outcome linkage — document whether the analysis connects teacher distribution to student achievement data, or explicitly states the absence of such linkage.
- Confound controls for school resources, student background, and community wealth — teacher-distribution gaps may reflect broader resource inequality.
- Comparison with at least one published deployment-equity study for a similar low-income country context.
- Teacher-absence context — document how effective teaching time is reduced by absence where absence data is available.

## Policy-Framing Requirements

No deployment-equity recommendation may be merged without:

- Non-punitive-transfer framing — explicitly state that deployment maps must not be used for punitive transfers without addressing incentive structures.
- Systemic-factor documentation — document that rural teacher gaps reflect systemic factors (compensation, infrastructure, career progression) not individual teacher failure.
- Non-use case documentation (not for punitive teacher transfers, not for stigmatizing rural teachers, not for treating qualification as equivalent to effectiveness).
