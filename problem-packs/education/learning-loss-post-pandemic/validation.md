# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Learning loss baselines must document:

- Assessment type, instrument, language, and administration date.
- Pre-pandemic and post-pandemic comparison methodology.
- Sample size and representativeness.
- Score metric and scaling methodology.
- Uncertainty bounds and comparability caveats.

## Model Or Signal Requirements

No learning loss estimate may be called validated until it includes:

- Data-quality tier assignment with criteria justification.
- Explicit uncertainty range.
- Sensitivity to alternative estimation methods.
- Statement of what the estimate is not (individual diagnosis, budget allocation, curriculum mandate).
- Cohort-stigmatization safeguards.
- Replication status for all quantitative estimates.
