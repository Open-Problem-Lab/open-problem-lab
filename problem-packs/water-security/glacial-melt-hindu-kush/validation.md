# Validation

## Validation Layers

| Layer           | Gate                           |
| --------------- | ------------------------------ |
| Structure       | `pnpm validate` passes schemas |
| Evidence        | Every claim has dated record   |
| Reproducibility | Rerunnable steps included      |
| Review          | Required reviewer approval     |
| Replication     | Quantitative claims replicated |

## Baseline Requirements

Melt projections must document: model ensemble, emission scenario, time horizon, basin-scale downscaling methodology, uncertainty ranges.

## Model Requirements

No risk ranking may be validated without: explicit methodology, sensitivity to assumptions, scenario documentation, replication status.
