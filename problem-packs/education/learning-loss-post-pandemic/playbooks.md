# Playbooks

## Analyst Playbook

1. Start with `problem.json`, `evidence.json`, and `tasks.json`.
2. Select one scoped task.
3. Add evidence before adding claims.
4. Keep quantitative outputs reproducible.
5. Mark every policy-relevant inference as reviewed, replicated, or blocked.

## Reviewer Playbook

Ask five questions:

1. What learning loss claim is being made?
2. What assessment data supports it, and from when?
3. What reconstruction method was used?
4. What would make the loss estimate wrong by a factor of two?
5. Who could misuse a learning loss estimate to blame teachers or cut education funding?

## Field-Reality Playbook

Do not ask whether learning loss is real — it is. Ask whether an education ministry official would use this estimate to change their remediation budget, teacher deployment, or remedial program design, and what would happen if the loss estimate was substantially wrong. Learning loss communication can create defeatism about the affected cohort — every output must clearly separate measurement from stigmatization.
