# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sanitation coverage baselines must document: survey source and year, sanitation facility type classification, usage-versus-access distinction, geographic grain, sample size and weighting methodology.

## Model Or Signal Requirements

No health-outcome correlation may be called validated until it includes: confounder control methodology, explicit non-causal caveats, sensitivity to alternative model specifications, replication status for all quantitative results.
