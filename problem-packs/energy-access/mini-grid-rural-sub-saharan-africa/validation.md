# Validation

| --------------- | ----------------------- |
| Structure       | `pnpm validate`         |

| Reproducibility | Rerunnable              |
| Review          | Approvers               |

## Baseline Requirements

Viability factors must document: data source, resolution, methodology, known gaps.

## Model Requirements

No viability ranking validated without: explicit criteria weighting, sensitivity analysis, non-investment caveats, replication.
