# Validation

| Layer           | Gate                    |
| --------------- | ----------------------- |
| Structure       | `pnpm validate`         |
| Evidence        | Dated records           |
| Reproducibility | Rerunnable              |
| Review          | Approvers               |
| Replication     | Quantitative replicated |

## Baseline Requirements

SAM burden must document: survey source and year, anthropometric protocol, sample size and design effect, confidence intervals, population denominator.

## Model Requirements

No coverage gap validated without: burden-treatment data reconciliation, HMIS completeness caveats, replication.
