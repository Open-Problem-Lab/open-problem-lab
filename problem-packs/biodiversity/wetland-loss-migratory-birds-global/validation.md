# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Flyway-scale wetland-loss analyses must document:

- Satellite wetland-change product used (JRC Global Surface Water) with resolution, temporal coverage, and wetland-type classification limitations.
- Tidal-cycle handling methodology for intertidal stopover sites.
- Small-wetland detection limitations — explicitly state the minimum detectable wetland size.
- Flyway monitoring-coverage assessment — document IWC and eBird data gaps for the analyzed flyway.
- Stopover-site selection rationale and spatial representativeness.

## Wetland-Bird Correlation Requirements

No wetland-loss-to-bird-decline attribution claim may be called actionable until it includes:

- Site-level wetland-change and bird-population data pairing methodology.
- Monitoring-coverage documentation stating which critical stopover sites lack bird-population data.
- Comparison with at least one published flyway-scale bird population trend study.
- Sensitivity analysis showing how correlation changes under alternative site-selection or temporal-window assumptions.
- Multi-threat attribution documentation listing at least three non-wetland threats to the analyzed species.

## Conservation-Priority Requirements

No conservation-priority recommendation based on wetland-bird analysis may be merged without:

- Multi-threat context documentation — wetland loss as one driver among many.
- Economic and social context for wetland conversion at analyzed sites.
- Explicit statement that correlation does not establish wetland loss as the primary driver of decline without experimental or natural-experiment evidence.
- Non-use case documentation (not for flyway-scale population attribution without multi-threat context, not for conservation-priority changes without stakeholder consultation).
