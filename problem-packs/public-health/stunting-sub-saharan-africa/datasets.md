# Dataset Inventory

| Source                      | Grain                      | Status             | Use                                                           |
| --------------------------- | -------------------------- | ------------------ | ------------------------------------------------------------- |
| DHS survey data             | Cluster/district, periodic | Usable             | Stunting prevalence, dietary diversity, intervention coverage |
| MICS survey data            | Cluster/district, periodic | Usable             | Supplementary stunting data for countries without recent DHS  |
| JME country estimates       | Country, annual            | Usable for framing | National trend context                                        |
| Osgood-Zimmerman 5km map    | 5km, 2000-2015             | Usable as baseline | Sub-national hotspot identification baseline                  |
| FEWS NET food security data | District, monthly          | Usable             | Food security shock context                                   |

## Required Dataset Properties

Date range, survey methodology, anthropometric measurement protocol, sample size, geographic grain, data-quality flags.

## Rejection Rule

A dataset is rejected for canonical modeling if methodology, date, or grain cannot be verified.
