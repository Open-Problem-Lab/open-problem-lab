# Dataset Inventory

| Source                 | Grain                  | Status | Use                               |
| ---------------------- | ---------------------- | ------ | --------------------------------- |
| Census migration data  | District, decadal      | Usable | Migration flows                   |
| MODIS NDVI             | 250m                   | Usable | Vegetation and crop condition     |
| CHIRPS rainfall        | 0.05 deg               | Usable | Rainfall anomalies                |
| NSS employment surveys | District, quinquennial | Usable | Economic drivers                  |
| Population census      | District, decadal      | Usable | Origin and destination population |

## Required Properties

Migration definition, census year, environmental data period, license.

## Rejection Rule

Rejected if migration or environmental data methodology unverifiable.
