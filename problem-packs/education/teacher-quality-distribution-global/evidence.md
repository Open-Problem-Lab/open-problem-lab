## Evidence Notes

### UNESCO Teacher Gap 2024

Use this source for the global teacher shortage baseline. The 44 million figure is a modeled projection based on enrollment targets, not an observed count. Teacher-qualification definitions vary by country — what counts as 'trained' differs across national systems. Deployment data is national aggregate — sub-national distribution is not systematically reported. Teacher absence is not captured in the shortage estimate. Always document these limitations in deployment-equity analyses.

### Service Delivery Indicator Surveys

Use SDI data for the strongest available within-country teacher-quality comparison. The 20-40 percent rural-urban gap is a range across countries. SDI measures teacher knowledge and skill, not actual classroom effectiveness — the relationship between assessment scores and student learning is imperfect. SDI coverage is limited to a few African countries. Always document the proxy-to-effectiveness limitation and geographic coverage gaps.

### Teacher Absence Rates

Use this source to document that teacher counts overstate effective teaching capacity. The 15-25 percent absence range is from a limited number of countries using unannounced visits. Absence measurement varies — some studies capture only physical absence, others include present-but-not-teaching. Rural absence may be higher due to longer commutes and fewer monitoring systems. Always present absence as a measured range with country-specific context.

### EMIS Data Quality Variability

Use this source to document the fundamental data constraint: EMIS data quality, update frequency, and public accessibility vary enormously across countries. Many countries do not publish sub-national teacher-distribution data. Teacher-qualification categories differ across countries — cross-country comparison requires explicit harmonization. Always document EMIS data quality for each analyzed country.

## Evidence Quality Rule

Evidence is not accepted because it sounds plausible. It is accepted when the source, method, limitations, and confidence are explicit enough for a reviewer to attack.
