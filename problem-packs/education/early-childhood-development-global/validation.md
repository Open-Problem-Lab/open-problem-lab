# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national ECD analyses must document:

- ECD data source (MICS ECDI, DHS) with instrument-comparability assessment across countries and survey rounds.
- Sub-national sample sufficiency — explicitly document whether district-level analysis is supported by sample sizes.
- Preschool-enrollment data source and quality-gap documentation.
- Nutrition-outcome data source for ECD context.
- Parenting-program coverage and evaluation-evidence assessment.

## ECD-Outcome Requirements

No ECD-outcome claim may be called actionable until it includes:

- Instrument-comparability documentation — explicitly state whether MICS ECDI scores are comparable across analyzed countries and survey rounds.
- Sub-national sample-sufficiency assessment — document whether cluster-level samples support district-level estimates.
- Parent-reporting limitation — document that ECDI scores are parent-reported milestones, not direct child assessment.
- Comparison with at least one published ECD-outcome analysis for a similar LMIC context.

## Intervention-Coverage Requirements

No ECD-effectiveness claim may be merged without:

- Coverage-vs-impact distinction — explicitly state that intervention coverage does not equal verified developmental impact without quality and dosage data.
- Quality-variation documentation — document whether preschool quality or parenting-program fidelity is measured.
- Scale-transferability caveat — explicitly state that parenting-intervention trial effects (0.3-0.5 SD) may not replicate at program scale.
- Sustainability documentation — explicitly state whether effects are sustained beyond program period.
- Non-use case documentation (not for claiming developmental impact from coverage data, not for overgeneralizing trial results to program scale).
