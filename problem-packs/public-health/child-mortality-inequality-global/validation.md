# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

District-level mortality estimates must document: survey source and year, mortality definition and recall period, sample size, confidence intervals, wealth-index construction, and small-area estimation methodology.

## Model Requirements

No inequality-trend claim validated without: multi-round comparison methodology, sample-size adequacy flags for districts with fewer than 500 live births, explicit non-stigmatization safeguards, and replication status.
