# Dataset Inventory

| Source                  | Grain            | Status              | Use                                     |
| ----------------------- | ---------------- | ------------------- | --------------------------------------- |
| DHS anthropometry       | Cluster/district | Usable              | Wasting and SAM prevalence              |
| SMART survey data       | District         | Usable, irregular   | Supplementary prevalence estimates      |
| HMIS SAM treatment data | District         | Usable with caveats | Treatment coverage numerator            |
| CMAM program data       | Facility         | Variable            | NGO and community-based treatment reach |
| Population estimates    | District         | Usable              | SAM burden denominator                  |

## Required Properties

Survey methodology, anthropometric protocol, treatment data completeness, population denominator, license.

## Rejection Rule

Rejected if methodology, date, or denominator unverifiable.
