# Dataset Inventory

| Source                   | Grain                 | Status | Use                                              |
| ------------------------ | --------------------- | ------ | ------------------------------------------------ |
| Landsat LST              | 30m, 1984-present     | Usable | Intra-urban heat island mapping                  |
| ERA5-Land temperature    | 0.1 deg, 1950-present | Usable | Background temperature and humidity              |
| WorldPop population      | 100m, annual          | Usable | Population exposure and density                  |
| DHS vulnerability        | Cluster/district      | Usable | Housing type, cooling access, elderly proportion |
| CHIRTS-daily temperature | 0.05 deg, 1983-2016   | Usable | Historical extreme temperature frequency         |

## Required Properties

Date range, geographic grain, temperature metric, population year, vulnerability indicators, license.

## Rejection Rule

Dataset rejected if grain, methodology, or license unverifiable.
