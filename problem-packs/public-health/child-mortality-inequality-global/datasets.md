# Dataset Inventory

| Source                     | Grain            | Status | Use                               |
| -------------------------- | ---------------- | ------ | --------------------------------- |
| DHS mortality data         | Cluster/district | Usable | Under-five mortality estimation   |
| DHS wealth-index data      | Household        | Usable | Wealth-quintile disaggregation    |
| MICS mortality data        | Cluster/district | Usable | Supplementary mortality estimates |
| Intervention-coverage data | Cluster/district | Usable | Coverage equity analysis          |
| WorldPop population        | 100m             | Usable | District population denominator   |

## Required Properties

Mortality definition and recall period, wealth-index construction, survey methodology, sample-size adequacy for district-level disaggregation, license.

## Rejection Rule

Rejected if survey methodology, sample size, or mortality definition unverifiable.
