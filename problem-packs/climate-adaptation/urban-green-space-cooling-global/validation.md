# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national urban green-space cooling analyses must document:

- Satellite LST product used (Landsat, MODIS) with resolution, temporal coverage, and LST-to-air-temperature validation status for the specific climate zone.
- Green-space classification methodology — document which types (parks, street trees, green roofs, community gardens) are distinguished.
- Irrigation-status documentation — explicitly state whether cooling-effect estimates account for irrigation availability.
- Informal-settlement data availability and green-space deficit assessment.
- Ground weather-station data availability for LST-to-air-temperature validation.

## Cooling-Effect Requirements

No green-space cooling-effect claim may be called actionable until it includes:

- LST-to-air-temperature transfer function documentation with climate-zone applicability assessment.
- Comparison with at least one published cooling-effect study for a similar urban context, with explicit LMIC-transferability assessment.
- Green-space-type differentiation — document which types contribute most to cooling in the analyzed city.
- Irrigation-dependence assessment — document whether cooling predictions assume irrigation and what happens without it.
- Sensitivity analysis showing how cooling estimates change under alternative LST-to-air-temperature assumptions.

## Vulnerability-Mapping Requirements

No heat-vulnerability recommendation based on green-space cooling analysis may be merged without:

- Informal-settlement green-space deficit documentation relative to formal residential areas.
- Green-gentrification risk assessment — document whether green-space improvement could displace vulnerable populations.
- Water-scarcity and irrigation-feasibility assessment for the analyzed city.
- Non-use case documentation (not for green-gentrification justification, not for irrigation-dependent cooling promises in water-scarce cities).
