# Validation

| Layer           | Gate                           |
| --------------- | ------------------------------ |
| Structure       | `pnpm validate`                |
| Evidence        | Dated records                  |
| Reproducibility | Rerunnable steps               |
| Review          | Required approvers             |
| Replication     | Quantitative claims replicated |

## Baseline Requirements

Heat exposure maps must document: satellite source, LST derivation method, ground-station validation, temporal compositing, uncertainty quantification.

## Model Requirements

No risk map validated without: vulnerability methodology, weighting sensitivity, non-stigmatization caveats, replication status.
