# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national fertilizer-access analyses must document:

- Fertilizer-use data source with sub-national granularity assessment — explicitly state whether national aggregate or district-level.
- Distribution-infrastructure barrier documentation — identify transport costs, price markups, and rural availability.
- Soil-nutrient data source (AfSIS, national surveys) with coverage and interpolation-limitation documentation.
- Fertilizer-response variability — document soil-specific response rates and the 30-50 percent degradation in depleted soils.
- Country-specific context — explicitly document that continental averages (15 kg/ha) mask enormous variation.

## Fertilizer-Access Mapping Requirements

No fertilizer-access claim may be called actionable until it includes:

- Sub-national granularity — explicitly state whether data is national aggregate or district-level.
- Distribution-infrastructure documentation — document transport-cost barriers and price-markup patterns.
- Data-gap documentation — state where sub-national fertilizer-use data is absent.
- Comparison with at least one published fertilizer-access analysis for a similar SSA context.

## Soil-Response Requirements

No fertilizer-effectiveness claim may be merged without:

- Soil-specific response variability — document that degraded soils show 30-50 percent lower response rates.
- AfSIS coverage limitations — explicitly state that soil data is concentrated at research sites with spatial interpolation.
- Response-rate context — document that research-station results may not represent on-farm conditions.
- Non-use case documentation (not for blanket fertilizer subsidies without soil-specific analysis, not for ignoring distribution-infrastructure barriers).
