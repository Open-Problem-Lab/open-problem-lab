# Playbooks

## Analyst Playbook

1. Start with `problem.json`, `evidence.json`, and `tasks.json`.
2. Select one scoped task.
3. Add evidence before adding claims.
4. For satellite fire-detection analysis: always document cloud-cover gaps, resolution limitations, and the difference between fire count and fire intensity.
5. For source-apportionment analysis: always document which methodology you use (chemical tracer, receptor model, satellite-based) and compare results with at least one published study.
6. For health-impact analysis: always document confound controls for non-burning PM2.5 sources, seasonal co-occurrences (Diwali, winter inversion), and hospital data limitations.
7. Mark every operational inference as reviewed, replicated, or blocked.

## Reviewer Playbook

Ask five questions:

1. What claim is being made about agricultural burning's contribution to PM2.5?
2. What source proves it — and is it a source-apportionment study, satellite correlation, or assumption?
3. What method separates agricultural burning from co-occurring PM2.5 sources?
4. What would make it false — particularly if fire-detection missed fires, monitors were too sparse, or confound control was inadequate?
5. Who could misuse it — particularly to justify punitive farmer action without providing viable alternatives, or to deflect attention from larger non-burning sources?

## Field-Reality Playbook

Do not ask whether a source-attribution map is impressive. Ask whether it would change a named decision for a named user at a useful time, without creating worse incentives. For agricultural burning, ask specifically:

- Would a Punjab Pollution Control Board officer use this to target Happy Seeder subsidies more effectively, or would it be used to justify blanket burning bans without addressing adoption barriers?
- Would a district agricultural officer use this to time extension outreach, or would it blame farmers for a systemic problem?
- Would a Delhi health department use this to issue targeted advisories, or would it ignore the larger non-burning PM2.5 burden?

The answer must include explicit non-use cases before any field-facing output is published.
