# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national mangrove-loss analyses must document:

- Satellite product used (GMW, Landsat, Sentinel-2) with resolution, temporal coverage, and detection limitations for narrow fringing mangroves.
- Pre-1996 baseline estimation methodology and uncertainty.
- Deforestation driver classification method and ground-verification status.
- Country-level data availability assessment.
- Administrative boundary version and sub-national aggregation method.

## Protection-Loss Requirements

No coastal-protection loss claim may be called actionable until it includes:

- Storm-surge attenuation relationship documentation with source field studies.
- Mangrove-width and density data used to parameterize the attenuation model.
- Bathymetry data quality assessment and model uncertainty bounds.
- Population-at-risk methodology with coastal-settlement type documentation.
- Comparison with at least one published protection-loss study for a similar coastline.
- Sensitivity analysis showing how protection values change under alternative attenuation assumptions.

## Driver-Attribution Requirements

No deforestation-driver attribution claim may be merged without:

- Driver classification methodology documentation (aquaculture, agriculture, development).
- Economic context for mangrove conversion — document livelihood dependencies.
- Ground-verification status for driver classification at deforestation sites.
- Non-use case documentation (not for aquaculture-ban justification without livelihood alternatives, not for individual-farmer culpability).
