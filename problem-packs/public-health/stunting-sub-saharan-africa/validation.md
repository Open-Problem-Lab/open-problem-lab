# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and completeness checks               |
| Evidence        | Every claim has dated evidence record                                |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Stunting estimates must document: survey source and year, anthropometric protocol and growth standard, sample size and design effect, confidence intervals, survey-round comparability notes.

## Model Requirements

No hotspot classification may be called validated until it includes: multi-round methodology, uncertainty quantification, data-insufficient district identification, replication status.
