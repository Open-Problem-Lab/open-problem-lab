# Dataset Inventory

| Source                | Grain             | Status                 | Use                                  |
| --------------------- | ----------------- | ---------------------- | ------------------------------------ |
| NOAA CRW DHW          | 5km, daily        | Usable                 | Thermal stress alerts                |
| AIMS LTMP surveys     | Per reef, annual  | Usable as ground truth | In-water bleaching and coral cover   |
| GBRMPA aerial surveys | Reef, per event   | Usable                 | Bleaching severity at scale          |
| Sentinel-2 imagery    | 10m, 2015-present | Usable                 | Benthic cover change detection       |
| IMOS SST              | ~2km, daily       | Usable                 | Higher-resolution SST for reef-scale |

## Required Properties

Date, resolution, validation methodology, license.

## Rejection Rule

Rejected if methodology or resolution unverifiable.
