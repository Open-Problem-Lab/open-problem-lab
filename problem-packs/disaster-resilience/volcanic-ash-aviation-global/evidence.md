## Evidence Notes

### ICAO VAAC Coverage 2024

Use this source for the global volcanic monitoring baseline. VAAC coverage maps show advisory responsibility, not actual detection capability — a VAAC can issue advisories even when detection infrastructure is sparse. Ground-based monitoring density varies within VAAC regions. Submarine volcanic monitoring is particularly sparse. Do not equate VAAC coverage with detection capability.

### Satellite Ash Detection Limitations

Use satellite ash detection with the critical caveat that the split-window technique cannot reliably distinguish ash from ice clouds, dust, or smoke — false-positive rates of 20-40 percent are documented. Nighttime detection has reduced accuracy. Multi-layer cloud conditions further reduce accuracy. Fine ash is harder to detect than coarse ash. Always document these limitations in detection-capability assessments.

### Ash Transport Model Uncertainty

Use ash-transport models with the critical caveat that eruption-source parameters (ash height, mass loading, particle size) are rarely measured in real time. Uncertainty propagates non-linearly — small errors in ash height produce large errors in downwind concentration forecasts. Model validation uses satellite detection as ground truth, but satellite detection itself has 20-40 percent false-positive rates — the validation chain is circular. Document these uncertainties explicitly.

### Eyjafjallajökull Operational Lessons

Use this source for operational context. The transition from zero-tolerance to risk-based thresholds was driven by economic pressure — the scientific basis for safe ash-concentration thresholds remains uncertain. Engine damage depends on ash composition, particle size, exposure duration, and engine type — no single threshold captures this complexity. Different aviation authorities apply different frameworks — there is no global consensus.

## Evidence Quality Rule

Evidence is not accepted because it sounds plausible. It is accepted when the source, method, limitations, and confidence are explicit enough for a reviewer to attack.
