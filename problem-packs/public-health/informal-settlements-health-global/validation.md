# Validation

| Layer           | Gate                    |
| --------------- | ----------------------- |
| Structure       | `pnpm validate`         |
| Evidence        | Dated records           |
| Reproducibility | Rerunnable              |
| Review          | Approvers               |
| Replication     | Quantitative replicated |

## Baseline Requirements

Settlement boundaries must document: mapping methodology, accuracy assessment, data currency, caveats about rapidly changing areas.

## Model Requirements

No access gap validated without: equity comparison, eviction-risk safeguards, replication.
