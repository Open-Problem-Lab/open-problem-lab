# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national clean-cooking analyses must document:

- ISO/IWA standards documentation — explicitly state which performance tiers are referenced.
- Lab-vs-field performance documentation — explicitly state whether data comes from laboratory or field measurements.
- Verification-system coverage — document what share of deployed stoves have independent compliance verification.
- Stove-category specificity — document which categories (improved biomass, LPG, ethanol, biogas, electric) are analyzed.
- Health-burden context — document WHO household air pollution mortality estimates with exposure-measurement caveats.

## Lab-Field Gap Requirements

No stove-performance claim may be called actionable until it includes:

- Lab-field gap methodology — document how laboratory and field measurements are compared.
- Stove-category-specific degradation estimates — document the 30-70 percent range with category breakdown.
- Degradation-factor documentation — identify fuel moisture, cooking practices, and maintenance as key drivers.
- Measurement-method comparison — document whether apparent degradation reflects actual performance loss or measurement-method differences.
- Long-term degradation assessment where data exists.

## Compliance-Verification Requirements

No compliance claim may be merged without:

- Verification-system coverage — document what share of deployed stoves have independent field verification (typically < 5%).
- Lab-certification vs. field-verification distinction — explicitly state that laboratory certification does not equal field-verified performance.
- Self-certification documentation — document whether manufacturers self-certify or use independent testing.
- Non-use case documentation (not for health-impact claims based on lab data alone, not for presenting compliance as field-verified performance).
