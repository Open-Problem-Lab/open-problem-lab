# Dataset Inventory

| Source                   | Grain                             | Status | Use                              |
| ------------------------ | --------------------------------- | ------ | -------------------------------- |
| ICIMOD glacier inventory | Individual glacier, 2010 baseline | Usable | Glacier area and volume baseline |
| GRACE/GRACE-FO mascons   | ~300 km, monthly                  | Usable | Regional mass-change trends      |
| ERA5-Land temperature    | 0.1 deg, 1950-present             | Usable | Melt-season temperature context  |
| CHIRPS precipitation     | 0.05 deg, 1981-present            | Usable | Precipitation trend analysis     |
| WorldPop population data | 100m, annual                      | Usable | Downstream population exposure   |

## Required Dataset Properties

Date range, geographic grain, melt-rate methodology, precipitation scenario, population estimate year.

## Rejection Rule

A dataset is rejected if grain, date range, methodology, or license cannot be verified.
