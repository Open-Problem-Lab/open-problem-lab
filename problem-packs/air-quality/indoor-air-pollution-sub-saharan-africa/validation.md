# Validation

| Layer           | Gate                    |
| --------------- | ----------------------- |
| Structure       | `pnpm validate`         |
| Evidence        | Dated records           |
| Reproducibility | Rerunnable              |
| Review          | Approvers               |
| Replication     | Quantitative replicated |

## Baseline Requirements

Exposure estimates must document: fuel classification, ventilation proxy, exposure model methodology, validation against measurements, fuel-stacking caveats.

## Model Requirements

No exposure map validated without: uncertainty ranges, fuel-stacking acknowledgment, replication.
