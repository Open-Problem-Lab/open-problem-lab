# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record                              |
| Reproducibility | Quantitative artifacts include rerunnable steps                      |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national environmental-migration analyses must document:

- Census migration-definition source and harmonization methodology — explicitly state which definition each country uses.
- Environmental indicator source, resolution, and temporal coverage.
- Economic confound variable availability and integration methodology.
- Cross-country comparability assessment — explicitly document definition differences and harmonization limitations.
- Decadal temporal resolution limitation — explicitly state that censuses cannot capture short-term or seasonal migration.

## Correlation-Analysis Requirements

No environmental-migration correlation claim may be called actionable until it includes:

- Economic confound controls — document which economic variables are controlled and their explanatory power.
- Variance-explained documentation — explicitly state what percentage of migration variance is explained by environmental vs. economic signals (typically 5-15 percent environmental).
- Spatial-autocorrelation handling — document whether and how spatial dependence in environmental and migration data is controlled.
- Comparison with at least one published environmental-migration correlation study for a similar South Asian context.
- Sensitivity analysis showing how correlation strength changes under alternative environmental indicators or confound specifications.

## Attribution Requirements

No climate-migration attribution claim may be merged without:

- Non-causal disclaimer — explicitly state that correlation does not imply causation and that environmental signals may proxy for agricultural income loss.
- Economic-driver primacy documentation — explicitly state that economic drivers explain more migration variance than environmental signals.
- Adaptive-vs-distress classification — document whether observed migration is likely adaptive (household strategy) or distress-driven (forced displacement), with evidence for the classification.
- Non-use case documentation (not for mobility restriction, not for climate-attribution claims without economic confound controls, not for presenting modest correlations as dominant climate-driven displacement).
