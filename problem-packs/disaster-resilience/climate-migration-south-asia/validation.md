# Validation

| Layer | Gate |
| ----- | ---- |
| Structure | `pnpm validate` |
| Evidence | Dated records |
| Reproducibility | Rerunnable |
| Review | Approvers |
| Replication | Quantitative replicated |

## Baseline Requirements

Migration data must document: census source and year, migration definition, origin-destination comparability.

## Model Requirements

No migration-environment claim validated without: economic confound controls, non-causal caveats, replication.
