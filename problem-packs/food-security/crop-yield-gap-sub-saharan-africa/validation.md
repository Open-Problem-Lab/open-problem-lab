# Validation

| Layer           | Gate                    |
| --------------- | ----------------------- |
| Structure       | `pnpm validate`         |
| Evidence        | Dated records           |
| Reproducibility | Rerunnable              |
| Review          | Approvers               |
| Replication     | Quantitative replicated |

## Baseline Requirements

Yield estimation must document: satellite source, ground-truth data, model methodology, error metrics.

## Model Requirements

No yield gap validated without: attainable yield definition, constraint attribution, replication.
